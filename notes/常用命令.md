
# 常用命名汇总

1. rebuild 内核
1.1 在buildroot 目录下面执行一下命名
```shell

make linux-rebuild
编译完成后会在 out/images目录下面生成 zImage、sun8iw20p1-t113-100ask-t113-pro.dtb 等文件
cd output/images
# 生成boot.img
mkbootimg --kernel zImage  --ramdisk  ramdisk.img --board sun8iw20p1 --base  0x40200000 --kernel_offset  0x0 --ramdisk_offset 0x01000000 -o  boot.img

# 推送 boot.img 与  sun8iw20p1-t113-100ask-t113-pro.dtb 文件到开发板
 mkdir /boot
 mount /dev/mmcblk0p4 /boot
 
adb push boot.img /boot/
adb push sun8iw20p1-t113-100ask-t113-pro.dtb /boot/
# 重启开发板生效
adb reboot
```
